# 🤖 Chatbot de Álgebra con IA

Aplicación web desarrollada con Next.js que funciona como un chatbot educativo de álgebra, capaz de generar respuestas mediante inteligencia artificial ejecutada localmente con Ollama.

---

## 🚀 Características

- Interfaz web interactiva  
- Generación de respuestas con IA local  
- Uso de base de datos con Prisma + SQLite  
- Arquitectura fullstack con Next.js  
- Persistencia de datos  
- Funcionamiento sin depender de APIs externas  

---

## 🧠 Tecnologías utilizadas

- Next.js  
- React  
- Prisma ORM  
- SQLite  
- Tailwind CSS  
- Node.js  
- Ollama  
- Llama 3
